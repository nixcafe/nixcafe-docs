<div class="hero">
  <h1>nixcafe</h1>
  <p>
    Nix scaffolding tools and frameworks for modern development.
    Opinionated tooling for reproducible builds, automatic flake discovery, and composable Nix modules.
    Start from a single config, not a hundred lines of Nix.
  </p>
</div>

<hr class="rule" />

<div class="section">
  <h2>Projects</h2>

  <div class="project">
    <h3>purr</h3>
    <p>
      A Nix flake framework with automatic discovery. Supports both <strong>flake-parts</strong> imports and direct <strong>mkFlake</strong> usage. Comes with a rich module system for composing dev shells, checks, packages, and more.
    </p>
    <a href="https://purr.nixcafe.org">purr.nixcafe.org</a>
  </div>

  <div class="project mt-3">
    <h3>develop-templates</h3>
    <p>
      A curated collection of Nix flake project templates. Bootstrap reproducible dev environments for Node.js, Python, Rust, Java, Kotlin, and more with a single command.
    </p>
    <p class="cmd">nix flake init -t beans#node</p>
    <a href="https://templates.nixcafe.org">templates.nixcafe.org</a>
  </div>
</div>
