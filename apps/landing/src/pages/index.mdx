# nixcafe

**Nix scaffolding tools and frameworks for modern development.**

nixcafe builds tools that make Nix more approachable — opinionated scaffolding, automatic flake discovery, and composable module systems.

---

## Projects

### [purr](https://docs.nixcafe.org/purr)

A Nix flake framework with automatic discovery. Supports both **flake-parts** imports and direct **mkFlake** usage. Comes with a rich module system for composing dev shells, checks, packages, and more.

---

## Philosophy

- **Zero boilerplate** — Start from a single config, not a hundred lines of Nix
- **Automatic discovery** — Your project structure defines your flake
- **Composable** — Mix and match modules like Lego blocks
- **Nix native** — Built on Nix fundamentals, no lock-in
